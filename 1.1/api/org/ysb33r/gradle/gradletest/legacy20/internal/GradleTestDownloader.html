Package: org.ysb33r.gradle.gradletest.legacy20.internal

[Groovy] Class GradleTestDownloader

        • org.ysb33r.gradle.gradletest.legacy20.internal.GradleTestDownloader

        • The downloader is a background task which will download Gradle distributions to a local folder in the build directory. This task is only enabled if the gradle job was not run offline. In the current implementation it will only look for 'bin' variants.
